🧮 FCN 模擬與評分

4️⃣ FCN 模擬層

📁 m7_fcn_simulation_output.json

模擬空間
KI ∈ {50, 55, 60}
Gap ∈ {10, 15, 18, 20}
→ 共 12 組情境

FCN總分
= [0.6 × Worst-of 基本股票分數 + 0.4 × Avg(基本股票分數)]
+ Avg(今日分數)
+ [0.3×KI分數 + 0.2×Gap分數 + 0.3×Tenor分數 + 0.6×Rate分數 + Type分數]

說明（非常重要）

第一段：股票本質

0.6 × Worst + 0.4 × Avg

反映 Worst-of 核心風險

判斷今天是否適合做 FCN

Avg(Today Score)
👉 判斷今天是否適合做 FCN

⸻

第三段：FCN條件（加權）
0.3 KI
0.2 Gap
0.3 Tenor
0.6 Rate
+ Type
+ 
👉 保留舊分數，但重新加權
