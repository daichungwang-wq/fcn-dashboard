📘 M7 Blueprint（系統邏輯說明）
1. 核心原則
- 先確保不會死，再去賺利息
- 只會接到最爛的那一檔
- 利率越高越要懷疑
2. 評分構成
估值 + 趨勢 + 結構 + 時機 + 資金 + 品質 + 類別調整
3. 技術面邏輯
- 年線 → 長期趨勢
- 6M / 3M → 中期結構
- 1W → 短期波動（找切入點）
4. 結構分類
- Pullback（最佳）
- Hot（偏熱）
- Top（做頭，避免）
- Downtrend（不做）
5. 曝險規則
Core：40%安全 / 50%警戒
Growth：25%安全 / 35%警戒
超過 → 警示但不擋單
6. 今日推薦邏輯
長期向上 + 中期回檔 + 曝險可控 → 🔥 今日推薦
